LDAP JavaBeans

These are examples of LDAP JavaBeans, included with the java LDAP classes. To run the samples, place the ldap class hierarchy under the html directory. They assume a Directory Server running on the same machine as the Web server, at port 389, and with the airius.ldif sample database distributed with the Netscape Directory Server.

Compile the source in this directory with "javac *.java".
 

  • LDAPGetProperty - an invisible bean to simplify getting a single property's values (from a single entry)
  • LDAPGetEntries - an invisible bean to get all DNs matching a given search filter, for example all "groups"
  • LDAPSimpleAuthenticate - an invisible bean that reports if a user can be authenticated
  • LDAPIsMember - an invisible bean that reports if a user is a member of a particular group
